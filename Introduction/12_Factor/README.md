K. Factor
time limit per test1 second
memory limit per test256 megabytes
You are given two integers N
 and F
. Your task is to check whether F
 is a factor of N
.

A number F
 is said to be a factor of N
 if N
 is divisible by F
.

Input
A single line containing two integers N
 and F
 (1≤N,F≤109)
.

Output
Print:

Yes if F
 is a factor of N
No otherwise
Example
InputCopy
36 6
OutputCopy
Yes
